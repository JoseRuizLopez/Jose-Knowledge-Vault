# Haystack

Framework de deepset para construir sistemas QA y [[RAG]] sobre documentos con soporte nativo para LLMs.

## Componentes

- **DocumentStore**: almacenamiento y recuperación (Elasticsearch, [[FAISS]], Weaviate).
- **Retriever**: recuperación densa o dispersa.
- **Generator**: generación de respuestas.
