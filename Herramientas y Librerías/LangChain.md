# LangChain

Framework de Python para construir aplicaciones con LLMs mediante cadenas, agentes, herramientas y memoria.

## Componentes

- **Chains**: secuencias de llamadas a LLMs.
- **Agents**: LLMs que deciden qué herramienta usar.
- **Retrievers**: integración con [[FAISS]], [[Pinecone]], etc. para [[RAG]].
- **Tools**: wrappers sobre APIs externas.

## Relación con [[LangGraph]]

LangGraph extiende LangChain para flujos de agentes con grafos de estado.
