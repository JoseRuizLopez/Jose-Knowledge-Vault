# LlamaIndex

Framework de Python especializado en ingesta, indexación y consulta de datos privados para LLMs, con enfoque en [[RAG]].

## Componentes

- **Data connectors**: ingesta desde PDFs, BBDD, APIs.
- **Index types**: VectorStoreIndex, KnowledgeGraphIndex, etc.
- **Query engines**: pipelines de recuperación y síntesis.

Más centrado en gestión de datos que [[LangChain]].
