# Prometheus

Sistema open-source de monitorización basado en series temporales con PromQL como lenguaje de consulta.

## Arquitectura

- **Scraping**: extrae métricas de endpoints HTTP periódicamente.
- **TSDB**: almacenamiento eficiente de series temporales.
- **Alertmanager**: gestión de alertas y notificaciones.

Habitualmente combinado con [[Grafana]] para visualización.
