# LangGraph

Librería sobre [[LangChain]] para agentes multi-paso con grafos de estado, ciclos, ramificaciones y persistencia.

## Conceptos

- **State**: diccionario compartido que fluye por el grafo.
- **Nodes**: funciones o LLMs que transforman el estado.
- **Edges**: conexiones condicionales entre nodos.
- **Checkpointing**: persistencia para reanudar ejecuciones.
