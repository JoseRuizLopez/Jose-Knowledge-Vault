# TensorBoard

Herramienta de visualización de TensorFlow para monitorizar experimentos ML en tiempo real.

## Funcionalidades

- Curvas de pérdida y métricas.
- Histogramas de pesos y gradientes.
- Proyector de embeddings (PCA, t-SNE).
- Dashboard de hiperparámetros.
