# Auditoría del Dataset

Proceso de examinar y validar un dataset para identificar problemas de calidad, sesgos y conformidad.

## Aspectos a auditar

- **Calidad**: nulos, duplicados, outliers, tipos incorrectos.
- **Distribución**: comparar con producción para detectar [[Data Drift]].
- **Sesgo**: desbalanceo de clases o grupos protegidos.
- **Linaje**: origen, transformaciones, permisos.

## Herramientas

[[Evidently AI]], Great Expectations, [[MLflow]].
