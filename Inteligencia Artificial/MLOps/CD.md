# CD — Continuous Delivery / Continuous Deployment

Prácticas MLOps que automatizan la validación y despliegue de modelos en producción.

- **Continuous Delivery**: modelo listo para producción con aprobación manual final.
- **Continuous Deployment**: despliegue automático si el modelo supera todas las pruebas.

## Componentes

Pipeline de validación, registro de modelos ([[MLflow]], [[Vertex AI]]), monitorización ([[Evidently AI]], [[Grafana]]).
