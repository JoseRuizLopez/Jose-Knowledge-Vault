# Online Machine Learning

Paradigma en el que el modelo se actualiza incrementalmente con cada nueva observación, sin reentrenar sobre el historial completo.

## Características

- Adaptable a distribuciones cambiantes ([[Data Drift]], [[Concept Drift]]).
- Bajo uso de memoria.
- Implementado en [[River (Python)]].
