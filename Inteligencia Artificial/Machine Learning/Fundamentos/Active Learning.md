# Active Learning

Paradigma en el que el modelo selecciona activamente las muestras más informativas para ser etiquetadas por un oráculo, reduciendo el coste de etiquetado.

## Estrategias

- **Uncertainty sampling**: muestra con menor confianza.
- **Query by committee**: mayor desacuerdo entre modelos.
- **Expected model change**: muestra que más cambiaría el modelo.
