# Data Augmentation

Técnicas que generan muestras adicionales aplicando transformaciones que preservan la etiqueta para reducir overfitting y mejorar la generalización.

## Técnicas comunes

- **Imágenes**: volteo, rotación, recorte, color jitter, Mixup, CutMix.
- **Texto**: back-translation, sustitución por sinónimos, EDA.
- **Audio**: time stretching, pitch shifting, ruido.
